# Frontend

Simple React/Vite dashboard for the exam intelligence workspace.

## What is here

- A PYQ dashboard with clickable question cards.
- A detail panel showing importance, accuracy, and other per-question analytics.
- An analytics tab for summary signals and placeholder charts.
- An agent tab with the intended prompt workflow.

## Run locally

```powershell
cd frontend
npm install
npm run dev
```

## Notes

- The UI uses local mock data for now.
- The download button is a placeholder for a future backend PDF endpoint.
- The agent tab is a UI shell for the future agent backend.
